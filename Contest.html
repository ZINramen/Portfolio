 Contest

공모전에 참여한 기록을 남겨둔 페이지입니다.

  2024년 만들래 10분 게임 콘테스트

 - 공모전 주제 : 10분 동안 플레이 가능한 단편작을 만들어 봅시다.

 - 공모전 접수 기간 : 2024.08.12 ~ 2024.08.23 (12일)

 - 참여 조건 : 30분 이내로 게임이 끝나야 하며 게임이 끝났음을 알려주는 요소가 있어야 합니다.


 - 1번째 참여 작품 : ION Adventure

 - 2번째 참여 작품 : Abrupt End

 - 참여 내용 : 개발한 게임의 게임 파일, 스크린샷 이미지를 공모전 사이트에 업로드하였습니다.

  게임 홍보를 위한 영상을 제작하여 사이트 내 게시 하였습니다.

 - 참여 이유 : 10분 분량의 프로젝트를 만드는 것은 어렵지 않다 생각했고, 게임 제작 능력을 기르기 위해 해당 공모전에 참여하였습니다.

 - 참여 후기 : 저는 10분이라는 시간 안에 게임 내 모든 내용을 즐길 수 있게 하는 방법에 대해 고민하며 제작할 프로젝트를 결정했습니다. 그래서 단순한 조작으로 플레이 가능한 게임을 만들었고, 간편하게 게임을 즐길 수 있도록 여러 플랫폼을 지원하는 과정을 진행했습니다.

 제작 과정에서 처음 써보는 기능과 라이브러리들을 사용하게 되어 어려운 점도 있었지만 새로운 내용을 배울 수 있어 좋았습니다.

 
해당 게임들은 공모전에선 좋은 결과를 얻진 못했으나 좋은 경험을 했다고 생각합니다.
만들어 보고 싶었던 게임을 제작해볼 수 있었고, 모바일 UI 배치나 화면 비율 조정 등을 개선하는 경험을 할 수 있었기 때문입니다.

모바일 플랫폼 포팅 경험은 추후 모바일 게임을 제작하는데에도 도움이 될것이라 생각합니다.
