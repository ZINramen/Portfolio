 2024년에 진행한 프로젝트

2024년에 제작한 게임 프로젝트를 모아둔 페이지입니다.

1. Outland (미완) (팀 개발)

- 게임 내용 : 맵을 진행하며 적을 상대하고, NPC와 상호작용해 스토리를 진행하는 게임이다.

- 사용툴 및 기술 : Unreal 5

- 제작기간 : 3개월

- 맡은 역할 : 프로그래밍(플레이어 전투, 데이터 테이블(스킬, 아이템) 연동, 적 및 보스 AI, 게임 내 UI 메뉴 등 구현)

- 전투 시스템 관련 기능에 대한 구현 코드를 일부 설명한 문서 : 다운로드

- 전투 시스템 내에 구현된 끼어들기에 대한 설명 문서 : 다운로드

- 영상 링크 : https://youtu.be/u5GIH84q90Y

- 스크린샷

2. TriAPO (팀 개발)

- 게임 내용 : 적들이 훔쳐간 전설의 무기 3개를 되찾기 위해 적들을 물리치는 플랫포머 액션 게임.
적을 쓰러뜨리고 손에 넣은 무기들은 같이 연계해서 사용 시 더 강력한 콤보를 사용할 수 있다.

- 사용툴 및 기술 : Unity, Aseprite(도트 및 컷신 제작)

- 제작기간 : 1.5개월(게임 개발) + 0.5개월(버그 수정 및 일부 컨텐츠 추가)

- 맡은 역할 : (주 역할)프로그래밍, 그래픽 / (부 역할) 기획


[1] 프로그래밍 : 플레이어 캐릭터 조작, 무기 교체 시스템, 무기 고유 능력(스킬, 필살기), UI 데이터 표시, 전투 관련 내용, 자동 저장, 도전 과제 시스템, 일부 적 AI, 최종 보스 2페이즈 제작

[2] 그래픽 : 캐릭터, 적, 보스, 배경, 컷신, 이펙트 등 제작, 최종 보스 Animation 제작

[3] 기획 : 게임 아이디어 기획, 게임 스테이지 일부 제작 등


- 무기 능력 및 교체 시스템 관련 코드 요약 문서 : 다운로드

- 영상 링크 : https://youtu.be/bRgg36j6OIA

- 스크린샷

3. VR 프로젝트 (미완) (팀 개발)

- 게임 내용 : 고정된 플레이어에게 다가오는 로봇들을 무기를 이용해 쓰러뜨리는 게임.

- 사용툴 및 기술 : Unity, XR Plugin

- 제작기간 : 4일

- 맡은 역할 : 프로그래밍(플레이어 단순 조작 및 VR 연동, NPC 및 적 구현 )

- 스크린샷

4. 커비 AR 프로젝트 (팀 개발)

- 사용툴 및 기술 : Unity, Vuforia

- 제작기간 : 3일

- 맡은 역할 : 프로그래밍(물리 엔진 관련 사항 조절, 보스 몬스터 AI, 몬스터 AI (클리어 형), 게임 플레이 UI 추가)

- 프로젝트 요약 문서 : 다운로드

- 영상 링크 : https://youtu.be/8hZe5maa2Oc?feature=shared

- 스크린샷

5. Abrupt End (팀 개발)

- 게임 내용 : 유튜브 광고로 자주 나오는 게임을 참조해 만들어 본 프로젝트.

- 사용툴 및 기술 : Unity

- 제작기간 : 4일

- 맡은 역할 : 프로그래밍(플레이어 조작, 맵 자동 생성, 상자 내 아이템 선택, 데이터(아이템 정보) 연동, 무기 고유 능력, 능력치 시스템)

- 스크린샷

- 게임 내용 : 목적지까지 빠르게 도달해야 높은 점수를 얻는 플랫포머 액션 게임.
